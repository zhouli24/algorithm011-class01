学习笔记

在学习总结中，写出不同路径 2 这道题目的状态转移方程。
1)、首先考虑无障碍物的情况：
状态方程，2条边（最上和最左）
dp[0][j] = 1;
dp[i][0] = 1;
其余点；dp[i][j] = dp[i-1][j] + dp[i][j-1];

2)、现在考虑有障碍物的情况：
状态方程，2条边（最上和最左），要考虑前面一个点的情况和当前点的情况，
dp[0][j] = (dp[0][j-1]==0 || map[0][j] == 1) ? 0 : 1;
dp[i][0] = (dp[i-1][0]==0 || map[i][0] == 1) ? 0 : 1;

其余点：
if (map[i][j] != 1) {
	dp[i][j] = dp[i-1][j] + dp[i][j-1];
} else {
	dp[i][j] = 0;
}
